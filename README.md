# virual-reality-tech
2025年第一学期虚拟现实建模与可视化实习安排 

一、地形数据三维可视化（GRID DEM）
1. 数据源        
	① QQ群上传的DEM文件：test.asc  DEM文件\
	② 自己从网站上下载一块高分辨率影像数据（如：https://www.earthol.com/ 或者 天地图 网站上）
2. 技术要求\
（1）基于MFC + OpenGL，编写单视窗文档，初始化OpenGL环境（蓝色底色）；（5分）\
（2）读取并且三维可视化DEM地形，分别采用正射投影和透视投影模式；（10分）\
（3）正射投影和透视投影下，可以中心旋转三维地形，可以拉近拉远地形，可以进行地形高度夸张；（10分）\
（4）自己从网站上下载一块高分辨率影像数据和矢量地图栅格化数据，作为地形的纹理进行渲染。（15分）
3. 时间要求\
	第三次实习课前上交源代码（原则上不接受后续补交）；
4. 成果评定\
		实习单人进行，占实习总分的40%；

二、室内三维建模与可视化
1. 数据源\
（1）武汉大学信息学部星湖实验大楼 DWG 设计数据\
（2）每个小组 负责两个楼层的建模
2. 技术要求\
	（1）可以使用3DMAX 、SKECHUP等大家熟悉的各种不同的建模工具，构建室内三维模型，上交格式一律统一为OBJ格式。要求具有建筑结构表达完整细致如走廊、门、窗户、墙 楼梯等；具有合适的纹理对功能区加以区分。考察的核心:室内语义的建模表达（20分）\
	（2）三维模型与二维矢量图形保持1:1比例对齐；（10分） 考察核心:三维模型与矢量数据的匹配性\
  （3）利用VS + OpenSceneGraph 将自己建模的OBJ模型、二维矢量数据叠加在第一次实习的三维地形数据上，可以进行漫游、缩放、旋转。考察核心:多源数据的三维场景集成可视化（20分）\
	（4）能够进行几何实体的拾取操作。考察核心 三维实体的几何查询方法 （10分）
3. 时间要求\
  2025年12月10日前提交模型、源代码和实习小结报告 （报告中包括小组成员分工、任务执行情况和程序运行截图）;
4. 成果评定\
  实习分组进行，本项实习占实习总成绩的60%；组长可以在报告中建议对没有认真完成分工任务的同学减分。
